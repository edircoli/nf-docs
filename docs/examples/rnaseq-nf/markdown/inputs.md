# Pipeline Inputs

This page documents all input parameters for the pipeline.

## Input/output options

### `--outdir` {#outdir}

**Type:** `string` | _Optional_ | **Format:** `directory-path`

The output directory where the results will be saved. You have to use absolute paths to storage on
Cloud infrastructure.

**Default:** `results`

### `--reads` {#reads}

**Type:** `string` | _Optional_

The input read-pair files

**Default:** `${projectDir}/data/ggal/ggal_gut_{1,2}.fq`

### `--transcriptome` {#transcriptome}

**Type:** `string` | _Optional_

The input transcriptome file

**Default:** `${projectDir}/data/ggal/ggal_1_48850000_49020000.Ggal71.500bpflank.fa`

### `--multiqc` {#multiqc}

**Type:** `string` | _Optional_

**Default:** `${projectDir}/multiqc`
